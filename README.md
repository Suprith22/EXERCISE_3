# Exercise 3 - Medical Image Classification using XGBoost

This exercise explores different XGBoost-based approaches for medical image classification using Chest X-ray and Brain Tumor MRI datasets.

Implemented approaches:

* Raw XGBoost
* PCA + XGBoost
* VGG16 + XGBoost

The goal was to compare traditional machine learning, dimensionality reduction, and deep feature extraction methods for medical imaging tasks.

## Folder Structure

### Chest_Xray_XGBoost

Implementation of XGBoost-based methods on chest X-ray images.

### Brain_Tumor_XGBoost

Implementation of XGBoost-based methods on brain tumor MRI images.

## Evaluation Metrics

* Accuracy
* F1-score
* Sensitivity
* Specificity

## Conclusion

This exercise demonstrated the effectiveness of combining deep CNN feature extraction with classical machine learning classifiers such as XGBoost for medical image classification.
